# Welcome

This is your new docs site, powered by [Docusaurus](https://docusaurus.io/).

Open the [MD Docs editor](https://mddocs-site.redbeach-67793768.westus3.azurecontainerapps.io) and ask the AI to make changes — they'll show up here after you publish.

## What's in this repo

| File / folder | Purpose |
| --- | --- |
| `docs/` | Every page on the site. One markdown file per page. |
| `docusaurus.config.js` | Site title, URL config, theme tweaks. |
| `sidebars.js` | Sidebar nav structure (auto-generates from `docs/` by default). |
| `.github/workflows/deploy-pages.yml` | The GitHub Action that publishes to Pages on each push to `main`. |

## Next steps

1. Replace this welcome page with your own content.
2. Add more pages in `docs/`. Use `sidebar_position` in frontmatter to control order.
3. Customize the site title in `docusaurus.config.js`.
